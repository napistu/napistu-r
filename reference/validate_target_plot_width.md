# Validate Target Plot Width

Validate Target Plot Width

## Usage

``` r
validate_target_plot_width(target_plot_width)
```

## Arguments

- target_plot_width:

  optional, specification for how large the planned plot is. knowing
  this helps to allow for more labels in a large plot where obscured
  labels will be filtered.

## Value

Invisible TRUE if validation passes, otherwise throws an error
