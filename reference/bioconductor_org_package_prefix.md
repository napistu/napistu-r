# Bioconductor Org Package Prefix

Find the prefix of the bioconductor package and its functions for a
species

## Usage

``` r
bioconductor_org_package_prefix(species)
```

## Arguments

- species:

  Species name

## Value

character prefix

## Examples

``` r
bioconductor_org_package_prefix("Homo sapiens")
#> org.Hs.eg
```
