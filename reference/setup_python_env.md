# Setup Python Environment

Setup Python Environment

## Usage

``` r
setup_python_env(napistu_config, verbose = TRUE)
```

## Arguments

- napistu_config:

  Napistu configuration object

- verbose:

  provide extra logging

## Value

A \`python_list\` containing Python modules and environment info

## Examples

``` r
if (FALSE) { # \dontrun{
    napistu_config <- create_napistu_config()
    setup_python_env(napistu_config)
} # }
```
