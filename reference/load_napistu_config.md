# Load Napistu Configuration from YAML

Reads a config containing a \`python_config\` and/or \`assets_config\`.

## Usage

``` r
load_napistu_config(file_path, verbose = TRUE)
```

## Arguments

- file_path:

  Path to YAML configuration file

- verbose:

  Logical scalar; if TRUE (default), prints informative messages about
  the current operation progress. If FALSE, runs silently.

## Value

Configuration object
