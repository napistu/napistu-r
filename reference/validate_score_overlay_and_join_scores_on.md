# Validate Score Overlay and Join Scores On

Validate Score Overlay and Join Scores On

## Usage

``` r
validate_score_overlay_and_join_scores_on(score_overlay, join_scores_on)
```

## Arguments

- score_overlay:

  optional, vertex-level scores containing \`score\` and the merging
  attribute specified in \`join_on\`

- join_scores_on:

  variable to use when merging vertices and score

## Value

Invisible TRUE if valid, throws error if invalid
