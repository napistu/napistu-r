# Select Score Overlay Palette

Create a palette for coloring nodes based on vertex-level scores.

## Usage

``` r
select_score_overlay_palette(score_palette, ...)
```

## Arguments

- score_palette:

  the name of the palette to use

  indication_scores

  :   A palette going from \[0,1\]

  log2 fold-change

  :   A symmetric blue-black-yellow palette centered on zero

- ...:

  extra arguments to pass into the color palette
